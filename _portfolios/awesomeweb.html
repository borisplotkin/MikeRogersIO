AwesomeWeb a simple freelance marketplace to help clients find and hire designers and developers for their projects.

AwesomeWeb approach me in January 2015 to help build out new features & improve current features.

Moving Enviroment

One of the key changes I made was moving from a single server enviroment to a PaaS enviroment. The main reasons behind this change was to allow for resouces to be focused on building out the application, and allow to allow easier scalability.

One of the tricky parts of the enviroment move was the assets, [I moved them to S3, make sure 301 redirects were in place from old URL]

We also moved our assets to be served via cloudfront which improved frontend performcne.
